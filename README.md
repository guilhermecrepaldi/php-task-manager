# PHP Task Manager

Simple to-do list with PHP and MySQL.

## Features
- User registration/login
- Create, complete, delete tasks
- Drag-and-drop reordering (SortableJS)
- Filter by status

## Requirements
- PHP 8.2+
- MySQL
- Apache/Nginx

## Setup
1. Import `schema.sql` into MySQL
2. Update credentials in `config.php`
3. Serve via `php -S localhost:8000` or deploy to Apache
