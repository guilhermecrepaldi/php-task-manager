# Notas do Projeto

## Stack
- PHP 8.2
- MySQL
- Sessões nativas PHP

## Estrutura
- `config.php` — conexão com banco e sessão
- `login.php` — autenticação (login + registro)
- `index.php` — listagem com filtros e drag-and-drop
- `adicionar.php` — cria nova tarefa
- `toggle.php` — alterna status
- `excluir.php` — remove tarefa
- `reordenar.php` — salva nova ordem via AJAX
